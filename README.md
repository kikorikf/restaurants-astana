# Рестораны Астаны

Личная база ресторанов Астаны с фильтрами, картой и отметками посещений.

## Возможности

- Карточки ресторанов с рейтингом 2ГИС, кухней, типом, чеком и адресом
- Фильтры по кухне, типу заведения, среднему чеку, статусу и факту посещения
- Карта на OpenStreetMap со всеми точками
- Отметка «была» и личная оценка 1–5 ★ (хранится в localStorage)
- Добавление нового места по ссылке 2ГИС (firmId парсится автоматически)
- Различие подборки Ростислава и собственных находок

## Технологии

- React 19 + Vite 5
- react-leaflet / Leaflet — карта
- localStorage — персональные данные без бэкенда

## Запуск

```bash
npm install
npm run dev
```

## Данные

Источник: подборка Ростислава + данные карточек 2ГИС (июль 2026).  
Поле `source: "rostislav"` — подборка Ростислава, `source: "own"` — добавлено вручную.
